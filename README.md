
# Fixed Rate Pig

This is a translation of the arcade game demo *Pig* to Ada and SDL2.

## Requirements

* Ada 2012 compiler
* GPRbuild
* [SDLAda](https://github.com/Lucretia/sdlada)

## Build and run

```sh
gprbuild fixed_rate_pig.gpr
cd resource
../binary/fixedratepig
```
